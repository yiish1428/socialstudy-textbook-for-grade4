1. 청주의 위치와 특성

2. 청주의 역사

직지의 고장 청주! 상당산성, 용두사지철당간 등 청주에 남아있는 역사적 장소와 문화유산을 알아봅니다.

직지와 인쇄 문화 상당산성 청주 역사

3. 청주의 생활과 변화

시간의 흐름에 따라 청주 사람들의 생활 모습이 어떻게 변화했는지 알아봅니다. 육거리시장, 무심천의 변화를 살펴봐요.

옛날 청주 도시 발전 현대 청주

4. 청주와 다른 지역의 교류

충북의 중심 청주와 다른 지역의 생활 모습을 비교하고, 지역 간 교류의 중요성을 알아봅니다.
